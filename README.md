### 🏟️ Project Type
- Data Analysis

The project investigates potential coverage gaps in a sports platform
by comparing the territorial sports supply with platform coverage data.

### 🎯 Project Goal
Identify regions and provinces that are not yet served (or only partially served)
by the platform, highlighting areas with greater potential for expansion and adoption.

### ❓ Main Business Question
Which Italian provinces show the largest gap between the territorial sports supply
and the coverage provided by a sports platform?

### 📊 Scope (MVP)
- Main unit of analysis: **province**
- Comparison between **registry supply** and **platform coverage**
- **Main KPIs**:
    - coverage ratio
    - coverage gap

### 🛠 Stack
- Python 3.13
- Pandas
- Matplotlib / Seaborn
- Jupyter
- Playwright (data collection)
- Looker Studio (dashboard)

### 📦 Data Sources
The analysis combines two data sources:
- An official sports registry providing entity counts by province
- A sports platform providing its current coverage by province

### 🗂️ Project Structure
```text
data/
  sources/
    sport_registries/<registry_name>/
      raw/                    # raw collected data
      processed/              # analysis-ready CSV/JSON
      quality/                # data quality checks
    sport_platforms/<platform_name>/
      raw/                    # raw platform data
      processed/              # analysis-ready CSV
  analysis/                   # notebook output
data_sample/                  # static sample data (committed)
notebooks/                    # Jupyter notebooks for EDA
src/
  config.py                   # centralized configuration
  data_collection/
    sport_registries/         # registry data collection pipeline
    sport_platforms/          # platform data processing
  utils/                      # shared utilities
reports/
  REPORT.md                   # analysis results and findings
  figures/                    # charts generated by notebooks
```

### 📈 Expected Outputs
- **Coverage gap ranking** by region and province
- **Coverage ratio heatmap** across provinces
- A geographic view highlighting areas where platform expansion
  could have the highest impact
